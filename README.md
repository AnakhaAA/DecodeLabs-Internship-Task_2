# Data Analytics Project 2

This project demonstrates exploratory data analysis using Python, Pandas, and Matplotlib.  
The dataset contains product sales information, and the analysis includes:

- Viewing dataset structure and summary statistics
- Calculating total and average sales
- Identifying product order counts
- Visualizing product sales with bar charts
- Highlighting highest order values

## Files
- `project2.py` → Python script with analysis code
- `Dataset for Data Analytics (1).xlsx` → Dataset used
- `Product Sales.png` → Visualization output (optional)

## Tools & Libraries
- Python 3.x
- Pandas
- Matplotlib

## Output Highlights
- Total Sales and Average Order values
- Product frequency distribution
- Bar chart visualization of product sales
- Top 5 highest orders
